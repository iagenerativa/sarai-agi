## âš ï¸ Known Issues

   - **Issue**: Proposals lost on restart
   - **Workaround**: JSON export/import before shutdown
   - **Fix Version**: v0.5

   - **Issue**: Once evolution executes, no automatic undo
   - **Workaround**: Manual rollback via admin tools
   - **Fix Version**: v0.6
